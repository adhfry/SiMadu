# INSTALL PACKAGE CHART ANNOTATION

Untuk chart yang sangat detail dengan garis alpha-cut, label zona, dan titik centroid, install:

```bash
npm install chartjs-plugin-annotation
```

Setelah install, refresh browser (Ctrl + Shift + R) untuk melihat chart yang lengkap.

## Fitur Chart Detail:

✅ Semua kurva membership function (Grade A, B, C)  
✅ Kurva hasil agregasi (clipped)  
✅ Garis vertikal di centroid  
✅ Garis horizontal untuk alpha-cut A, B, C  
✅ Label zona di atas chart (Sangat Rendah, Rendah, Standar, Tinggi, Sangat Tinggi)  
✅ Titik centroid dengan tanda silang merah  
✅ Titik-titik potong antar kurva  

**Penting:** Jangan lupa restart dev server setelah install!
